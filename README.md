

####### Running instructions #######
1. Copy the jar mobile-micro-service-0.0.1.jar 
2. Run the following command to run the application

java -jar mobile-micro-service-0.0.1.jar

####### Curl command for testing #######

1. curl --location --request GET 'http://localhost:8080/mobile-handset/mobile/search?announceDate=1999&priceEur=200&sim=Mini-SIM'
2. curl --location --request GET 'http://localhost:8080/mobile-handset/mobile/search?priceEur=200'
3. curl --location --request GET 'http://localhost:8080/mobile-handset/mobile/search?sim=eSim'
